# ASPNetCoreWebAPI-Angular-DEMO

1. Navigate to API folder
    1. Run **`dotnet restore`** command.
    1. Run **`dotnet ef database update`** command.
1. Navigate to AngularApp folder
    1. Run **`npm install`** command.
    1. Run **`ng build`** command.
